# 🔬 بوت المحتوى العلمي المتطور
### Science Channel Monitor Bot — v2.0.0

بوت تيليغرام متكامل لرصد القنوات العلمية، تصنيف المحتوى، وتوزيعه تلقائياً.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 🔍 كشف ذكي | يكشف المحتوى العلمي بدون هاشتاجات أو معها |
| 🌍 ثنائي اللغة | يدعم العربية والإنجليزية |
| 📤 تحويل تلقائي | يحوّل المحتوى العلمي لقناة الوجهة مع كامل التفاصيل |
| 📊 إحصائيات شاملة | إحصائيات لكل قناة وإجمالية |
| 🌐 منصات خارجية | يرسل للمواقع والمنصات عبر Webhook |
| ⚙️ لوحة تحكم | إعدادات تفاعلية بالأزرار |
| 🎚️ حساسية متغيرة | تحكم كامل في حساسية الكشف |
| 🔑 كلمات مخصصة | إضافة كلمات علمية خاصة بك |

## 🚀 خطوات التشغيل

### 1. المتطلبات
```bash
Python 3.10+
```

### 2. التثبيت
```bash
# استنساخ المشروع
git clone <repo_url>
cd science_bot

# تثبيت المكتبات
pip install -r requirements.txt
```

### 3. إعداد البوت
```bash
# انسخ ملف الإعدادات
cp .env.example .env

# افتح .env وأضف:
# BOT_TOKEN=<توكن بوتك من @BotFather>8693784263:AAFlgstAc5MICwwWmbIUOPeGUmeusV9-F5k
# ADMIN_IDS=<معرف حسابك في تيليغرام>8027320875
```

### 4. تشغيل البوت
```bash
python main.py
```

## 📋 قائمة الأوامر

### الأوامر الأساسية
```
/start          — تشغيل البوت
/help           — قائمة الأوامر
/stats          — الإحصائيات الشاملة
/settings       — لوحة الإعدادات التفاعلية
```

### إدارة القنوات
```
/addchannel    <معرف>    — إضافة قناة للمراقبة
/removechannel <معرف>    — إيقاف مراقبة قناة
/channels                — عرض جميع القنوات المراقبة
/chanstats     <معرف>    — إحصائيات قناة محددة
```

### قناة الوجهة
```
/setdest <معرف>    — تعيين القناة التي تُحوَّل إليها المحتويات
```

### المنصات الخارجية
```
/addplatform    <اسم> <webhook_url> [api_key]
/removeplatform <اسم>
/platforms
```

### الكلمات المفتاحية
```
/addkeyword    <كلمة>    — إضافة كلمة علمية مخصصة
/removekeyword <كلمة>    — حذف كلمة مفتاحية
```

### إدارة المشرفين
```
/addadmin    <معرف>    — منح صلاحيات مشرف
/removeadmin <معرف>    — سحب صلاحيات مشرف
```

## 🔧 كيفية الإعداد

### إضافة البوت كمشرف في قناة
1. افتح إعدادات قناتك في تيليغرام
2. اضغط "Administrators" ثم "Add Administrator"
3. ابحث عن اسم بوتك وأضفه
4. امنحه صلاحية **"Post Messages"** أو **"Read Messages"**
5. أرسل للبوت: `/addchannel <معرف_القناة>`

### الحصول على معرف القناة
- أضف **@userinfobot** للقناة وستحصل على المعرف
- أو أعد توجيه رسالة من القناة لـ @userinfobot

### إعداد الـ Webhook للمنصات
```bash
/addplatform موقعي https://yourdomain.com/api/science-posts YOUR_API_KEY
```

الـ Webhook سيستقبل JSON بهذا الشكل:
```json
{
  "source_channel": "اسم القناة",
  "channel_link": "https://t.me/channel",
  "content": "نص المنشور",
  "media_type": "text|video|photo|document",
  "categories": ["فيزياء", "كيمياء"],
  "keywords": ["ذرة", "جزيء"],
  "hashtags": ["#علوم"],
  "date": "الثلاثاء، 1 يناير 2025",
  "time": "10:30 ص",
  "score": 8
}
```

## 🔬 نظام الكشف عن المحتوى العلمي

### التصنيفات المدعومة
| التصنيف | أمثلة |
|---------|-------|
| فيزياء | كم، نسبية، جسيمات، موجات |
| كيمياء | جزيء، تفاعل، عناصر |
| أحياء | خلية، جينات، DNA، تطور |
| طب | لقاح، علاج، مناعة |
| رياضيات | تفاضل، جبر، إحصاء |
| فلك | مجرة، ثقب أسود، نجم |
| تقنية | ذكاء اصطناعي، برمجة، روبوت |
| بيئة | مناخ، طاقة متجددة |
| هندسة | أتمتة، روبوت، ميكانيكا |
| نانو/بيوتك | هندسة وراثية، نانو |

### مستويات الحساسية
| المستوى | الوصف |
|---------|-------|
| 1 | منخفض جداً — يصنّف أي محتوى قد يكون علمياً |
| 2 | منخفض — مناسب لمعظم القنوات |
| 3 | متوسط — موصى به |
| 4 | عالي — يشترط وجود عدة مؤشرات |
| 5 | عالي جداً — للقنوات المتخصصة جداً |
